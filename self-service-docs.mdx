# CRM Integration Self-Service

## What They Need

Ability to independently connect and manage their CRM system.

## Self-Service Features

- Connect/disconnect Salesforce account
- Connect/disconnect HubSpot account
- API key generation
- OAuth authentication management
- Field mapping configuration
- Contact sync settings
- Candidate data import/export
- Duplicate record handling
- Sync failure troubleshooting
- Integration health dashboard
